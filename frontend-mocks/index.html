Backend API Integration

Both interfaces connect to the same FastAPI backend with these endpoints:

  • POST /auth/login - User authentication
  • GET /templates - List available templates
  • POST /generate - Generate content from template
  • GET /generate/{job_id} - Check generation status
  • GET /providers/models - List available AI models
